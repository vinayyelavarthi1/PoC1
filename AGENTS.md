# AGENTS.md

Use Codex in this repository mainly for pull request review.

Basic rules:
- review only the pull request diff
- focus on meaningful issues in the changed code
- keep comments concise and practical
- be evidence-based and avoid guessing
- if there is no real issue, respond with `No significant findings.`

When relevant, check for:
- correctness issues
- security issues
- deployment or pipeline risks
- missing validation or error handling
- changes that may break existing behavior

Return markdown.
